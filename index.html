PayGreen UI

La librairie de composants UI officielle de PayGreen pour React 🎨

Commencer 🚀

Un style harmonisé, peu importe le projet

De notre site vitrine PayGreen IO aux back-offices ClimateKit et CharityKit en passant par notre FAQ, toutes nos applications React utilisent la librairie de composants PayGreen UI. 👩‍🎤

Un large choix de composants

Utilisable autant pour des vitrines que pour des back-offices de gestion, PayGreen UI décline ses composants en plusieurs versions visuelles : retrouvez notamment le composant Button pour les vitrines, et son homologue DaButton pour les interfaces de type DAshboard. 💪

Dark mode compliant

100% de nos composants sont disponibles à la fois en lightmode et en darkmode. Utilisez le Toggle 🌞🌜 en haut à droite pour vous en rendre compte !
